# Aleksander proxy

Android-приложение для поиска актуальных прокси для Telegram.

- Кнопка **«Найти прокси»** — получает 10 рабочих прокси.
- Переключатель типа: **MTProto** и **SOCKS5**.
- Каждый прокси проверяется на живость (TCP-подключение) и сортируется по скорости.
- У каждого прокси своя кнопка **копирования** — копируется готовая ссылка вида
  `https://t.me/proxy?...`, которая при вставке в браузер/Telegram сразу подключает прокси.

Современный интерфейс на **Jetpack Compose (Material 3)**, тёмная тема.

## Как получить готовый `.apk` (без установки чего-либо на компьютер)

Сборка идёт в облаке через **GitHub Actions**. Нужен только аккаунт на GitHub.

### 1. Создайте репозиторий
- Зайдите на https://github.com → **New repository**.
- Имя, например, `aleksander-proxy`. Можно приватный. Нажмите **Create repository**.

### 2. Загрузите файлы проекта
Проще всего через браузер:
- На странице пустого репозитория нажмите **uploading an existing file**.
- Перетащите **всё содержимое** папки `AleksanderProxy` (включая скрытые папки
  `.github` и файлы `.gitignore`, `build.gradle.kts` и т.д.).
- Нажмите **Commit changes**.

> Важно: папка `.github` со сборочным workflow обязательна — именно она собирает APK.

### 3. Дождитесь сборки
- Откройте вкладку **Actions** в репозитории.
- Запустится задача **Build APK** (2–4 минуты). Если не запустилась сама —
  откройте её и нажмите **Run workflow**.

### 4. Скачайте APK
- Когда задача станет зелёной ✓ — откройте её.
- Внизу в разделе **Artifacts** будет **`Aleksander-proxy-apk`**.
- Скачайте, распакуйте `.zip` — внутри `app-debug.apk`.

### 5. Установите на телефон
- Перекиньте `app-debug.apk` на Android-телефон.
- Откройте его и разрешите **«Установка из неизвестных источников»**.

## Сборка в Android Studio (альтернатива)
Откройте папку проекта в Android Studio → **Build → Build Bundle(s) / APK(s) → Build APK(s)**.
Готовый файл появится в `app/build/outputs/apk/debug/`.

## Источники прокси
- MTProto: `shablin/mtproto-proxy`, `SoliSpirit/mtproto`
- SOCKS5: `TheSpeedX/PROXY-List`, `hookzof/socks5_list`, `monosans/proxy-list`

Все списки публичные и обновляются автоматически. Бесплатные прокси нестабильны —
если какой-то не подключается, нажмите «Найти прокси» ещё раз.
